# Frontend

## Overview
This is the frontend folder for Shinsei-Keikaku, built with modern web technologies.

## Tech Stack
- **Framework**: React
- **Build Tool**: Vite
- **Styling**: CSS
- **State Management**: Context API

## Key Features
- Responsive design
- Component-based architecture
- User authentication
- Dynamic routing

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Project Structure
```
src/
├── assets/
├── components/
├── context/
└── Pages/
```

## Available Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run linter
